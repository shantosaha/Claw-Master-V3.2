### Primary Storage (By Topic)

Documents are stored in topic-based folders:

- **Machine/** - Machine and system-related documentation
  - `Implementation/` - Machine implementation plans
  - `Documentation/` - Machine feature documentation

- **Project/** - Project management documentation
  - `Planning/` - Project planning and summaries
  - `Analysis/` - Project analysis reports
  - `Architecture/` - Architecture documentation
  - `Operations/` - Deployment and operations
  - `Quality-Assurance/` - QA strategies and checklists
  - `Readiness/` - Market readiness documentation

- **Quality/** - Quality assurance documentation
  - `Audits/` - All audit reports
  - `Testing/` - Testing documentation

- **Development/** - Development documentation
  - `Implementation/` - Implementation plans
  - `Tasks/` - Task lists and tracking
  - `Workflows/` - Development workflows

- **Documentation/** - General documentation
  - `Guides/` - User and testing guides
  - `Handover/` - Project handover documentation
  - `General/` - README and general docs

- **Git/** - Git and version control documentation
  - `Setup/` - GitHub setup and instructions

### Functional Views (By Type)

Quick access to all documents of the same type across topics:

- **Views/Implementation/** - All implementation plans
- **Views/Tasks/** - All task lists
- **Views/Guides/** - All guides
- **Views/Audits/** - All audit reports
- **Views/Plans/** - All planning documents
